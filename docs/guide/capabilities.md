# 系統功能介紹

本文件說明 Barney's Notes 筆記管理系統的主要功能與特色，幫助您快速了解本系統能為您提供什麼服務。

[[toc]]

## 核心功能

### 筆記分類管理

系統提供完整的筆記分類架構，涵蓋多個技術領域：

- **前端開發**：前端相關技術筆記，包含實用開發技巧
- **JavaScript**：JavaScript 程式語言相關筆記
- **TypeScript**：TypeScript 型別系統與應用
- **CSS**：樣式設計與排版技巧
- **Library**：第三方函式庫使用指南
- **Docker**：容器化技術相關筆記
- **Git**：版本控制系統使用方法

### 疑難雜症彙整

專門收錄開發過程中遇到的問題與解決方案：

- 詳細記錄問題現象與發生情境
- 提供經過驗證的解決方法
- 附上相關參考資料與延伸閱讀
- 方便日後遇到類似問題時快速查閱

### 文件撰寫指南

提供完整的文件編寫規範：

- Markdown 格式編排標準
- 統一的文件結構與風格
- 確保內容一致性與可維護性
- 提升閱讀體驗與檢索效率

## 技術特色

### 基於 VitePress 建構

採用 VitePress 作為靜態網站生成器：

- 快速的頁面載入與切換
- 支援 Markdown 擴充語法
- 內建搜尋功能
- 響應式設計，適配各種裝置
- 支援深色模式切換

### 優質閱讀體驗

- 清晰的導航結構
- 自動生成文章目錄
- 程式碼語法高亮
- 支援自訂提示區塊（info、tip、warning、danger）
- 內建全文搜尋功能

### 便利的內容管理

- 使用 Markdown 格式撰寫
- Git 版本控制管理
- 支援線上編輯（GitHub）
- 自動部署更新
- 顯示最後更新時間

## 快速開始

### 線上瀏覽

直接造訪 [https://bugloop.com](https://bugloop.com) 即可開始瀏覽筆記內容。

### 本地開發

如需在本地環境執行或貢獻內容：

```sh
# 複製專案
git clone https://github.com/lllloo/note.git

# 安裝相依套件
npm install

# 啟動開發伺服器
npm run docs:dev
```

### 內容貢獻

歡迎透過以下方式參與：

- 回報問題或建議（GitHub Issues）
- 提交內容修正或新增（Pull Request）
